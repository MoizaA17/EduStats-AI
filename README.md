# 🎓 EduStats AI

EduStats AI is a simple student marks analysis web application built with **Python**, **NumPy**, and **Streamlit**. It allows users to enter students' marks and instantly generates useful statistics and insights through an interactive web interface.

---

[![Live App](https://img.shields.io/badge/🚀%20Live%20Demo-Streamlit-red?style=for-the-badge)](https://edustats-ai.streamlit.app/)

A simple student marks analysis web application built with **Python, NumPy, and Streamlit**.

## 🚀 Live Demo

👉 **Try the application here:**

https://edustats-ai.streamlit.app/

---

## 🚀 Features

- 📊 Calculate average marks
- 🏆 Find the highest marks
- 📉 Find the lowest marks
- 🥇 Display topper student index
- ✅ Count passed students
- ❌ Count failed students
- 📈 Calculate pass percentage
- 💻 Interactive Streamlit web interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/MoizaA17/EduStats-AI.git
```

Go into the project folder

```bash
cd EduStats-AI
```

Run the Streamlit application

```bash
streamlit run app.py
```
